# 🌟 Features

The Autonomous Research Assistant comes packed with a variety of powerful features designed to make your research process seamless, customizable, and efficient.

## 🎨 UI & User Experience
- **Dynamic Screen Modes**: Ability to change the screen mode in settings (Dark, Light, or System default) to suit your viewing preferences.
- **Real-Time Agent Status**: Real-time feedback showing which specialized agent is currently working.
- **Interactive Document Viewer**: View the raw document chunks retrieved, filterable by their source.
- **🗺️ Routing Map**: Transparency into why the Planner chose specific tools for every sub-query.
- **⏱️ Research Timing**: Displays the exact duration taken to generate the report and metrics on chunks processed.

## 💾 Export & Download
- **Export to Markdown**: Ability to download the final research report in a well-formatted Markdown (`.md`) file.
- **Export to Text**: Ability to download the generated report as a plain Text (`.txt`) file for easy sharing and compatibility.

## 🔍 Advanced Search & Ingestion
- **Local File Search**: Ability to search based on uploaded files. Ingest PDFs directly into a local ChromaDB instance to build your own local knowledge base without any cloud embedding costs.
- **Live Web Search**: Searches the live web using Tavily for up-to-date information.
- **Academic Database Search**: Queries ArXiv for academic papers and scholarly articles.

## 🤖 Model Customization & Integrations
- **Change Your Model**: Ability to change the LLM model to be used dynamically.
- **Bring Your Own Key (BYOK)**: Securely input and use your own API keys for the tools and models you prefer.
- **LangSmith Integration**: Native integration for monitoring, tracing, and debugging the multi-agent LLM application effectively.
- **OpenRouter Integration**: Access a wide variety of Free and Paid LLMs via the OpenRouter ecosystem.
